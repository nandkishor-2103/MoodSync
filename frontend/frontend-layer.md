## Frontend Layer Structure:

---

### 1. **UI Layer**
- UI dikhana aur navigation handel krna (pages, components)

### 2. **API Layer**
- Backend se communicate karne ke liye (services/auth.api.js)

### 3. **State Layer**
- data store krna ke liye (auth.context.js, posts.context.js)

### 4. **Hook Layer**
- State and API manage krna hote hai (hooks)
